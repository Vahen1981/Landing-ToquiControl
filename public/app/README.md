# ToquiControl GMC-001 Programmer

## Descripción

Esta aplicación web permite programar de manera fácil el dispositivo ToquiControl GMC-001. Proporciona una interfaz intuitiva para configurar modos de nota, canales MIDI, y construir acordes, simulando la pantalla LCD y controles del dispositivo.

## Requisitos

- Un navegador web moderno (Chrome, Firefox, Edge, etc.)
- El dispositivo ToquiControl GMC-001 debe estar conectado vía USB al computador para que la aplicación funcione correctamente. Sin la conexión USB, la app no podrá comunicarse con el dispositivo y no tendrá funcionalidad.

## Cómo usar

1. Abre la aplicación en tu navegador desde [https://vahen1981.github.io/AppGMC-web/](https://vahen1981.github.io/AppGMC-web/).
2. Conecta el dispositivo ToquiControl GMC-001 a tu computador vía USB.
3. La aplicación detectará automáticamente el dispositivo y mostrará su estado como "online".
4. Usa los controles en la interfaz para configurar:
   - Modo de nota (habilitar/deshabilitar)
   - Canal MIDI general
   - Construir acordes y notas
5. Envía los cambios al dispositivo usando los botones "Send".

## Características

- Simulación visual del LCD del GMC-001
- Control de LEDs y pedales
- Constructor de acordes
- Gestión de modos de nota y canales MIDI

## Notas

Esta aplicación está diseñada específicamente para el dispositivo ToquiControl GMC-001. Asegúrate de que el dispositivo esté correctamente conectado y reconocido por tu sistema operativo antes de usar la app.
